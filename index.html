Sobre la aplicación:

Esta aplicación hecha en Javascript permite adaptar el formato de cualquier texto para que este cumpla con las Normas APA. Ideal para informes, tesis, y otros documentos que requieran ete formato específico.

Conozca Tesseract.js

Tesseract.js es un port puro del popular motor Tesseract OCR a Javascript. Esta librería soporta más de 100 lenguajes, orientación automática de texto y detección de scripts, y una interfaz simple para detectar párrafos, palabras, y caracteres correctamente. Tesseract.js puede ejecutarse tanto en un buscador como en un servidor con NodeJS.
